## üé® IMAGE STYLE GUIDE

### Color Palette for Images
- **Primary:** Deep royal blue (#1e40af range)
- **Accent:** Electric cyan (#06b6d4 range)
- **Background:** Dark slate (#0f172a range) or off-white
- **Highlights:** Soft purple, amber for warmth

### Photography Style
- **Modern & Clean:** Minimal clutter, professional
- **Authentic:** Real people, real workspaces, candid moments
- **Diverse:** Include diversity in team photos
- **Well-lit:** Natural or professional lighting, no harsh shadows

### Illustration Style
- **Flat Design:** Clean, minimal vector illustrations
- **Isometric:** For process/workflow visualizations
- **Gradients:** Subtle blue-to-cyan gradients
- **Line Art:** Thin, elegant lines for icons

### Screenshot Style
- **Clean UI:** Remove sensitive data, use sample data
- **Consistent Theme:** Use same color scheme as website
- **High Resolution:** Crisp, readable text
- **Context:** Show UI in device mockups

## üîß TECHNICAL SPECS

### Image Optimization
- **Format:** WebP with JPEG fallback
- **Max Size:** 200KB per image
- **Hero Images:** Max 500KB
- **Lazy Loading:** All below-fold images
- **CDN:** Use Vercel Edge or Cloudinary

### Responsive Images
```tsx
<Image
  src="/images/hero"
  alt="Description"
  width={1920}
  height={1080}
  priority={true} // For hero only
  className="object-cover"
/>
```

6. **Add Alt Text:** SEO and accessibility matter
7. **Test on Mobile:** Ensure images look good on all devices
